# Sangam consensus

A panel of models answers in parallel and a synthesizer reconciles one best answer — built-in or a custom panel.

```bash
python recipes/10-sangam-consensus/main.py
```

Reads `BHARATROUTER_API_KEY` from the environment and points at `https://api.bharatrouter.com/v1`.
Set `BHARATROUTER_MODEL` to any id from [the catalog](https://bharatrouter.com/models), or let the
recipe pick one for you.

**Web version:** https://bharatrouter.com/cookbook/sangam-consensus
